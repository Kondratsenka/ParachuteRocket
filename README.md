# ParachuteRocket
Симулятор запуска ракеты с парашютом на Qt.  
Позволяет вводить параметры ракеты и следить за её высотой, скоростью и временем.

## Функционал

* Ввод параметров ракеты: масса, высота, сила парашюта.
* Симуляция полёта ракеты с парашютом.
* Таблица: время | высота | скорость.
* Визуализация траектории (опционально)
